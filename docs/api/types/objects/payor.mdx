Payors are used to track payor info that can be tied to other data objects in Pay Theory.


```graphql
type Payor {
  address_line1: String
  address_line2: String
  city: String
  country: String
  email: String
  full_name: String
  merchant_uid: String
  metadata(
    query_list: [QueryPair]
  ): AWSJSON
  payor_id: String
  phone: String
  postal_code: String
  region: String
}
```
