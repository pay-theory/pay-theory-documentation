Payment Method Tokens are meant to store info that represents a tokenized Bank Account, Credit Card, or Debit Card.


```graphql
type PaymentMethodToken {
  address_line1: String
  address_line2: String
  card_brand: String
  city: String
  country: String
  exp_date: String
  full_name: String
  is_active: Boolean
  last_four: String
  merchant_uid: String
  metadata(
    query_list: [QueryPair]
  ): AWSJSON
  payment_method_id: String
  payment_type: PaymentType
  payor(
    query_list: [QueryPair]
  ): Payor
  postal_code: String
  region: String
  wallet_type: WalletType
}
```




### Fields

#### [`address_line1`](#)<Bullet />[`String`](/docs/api/types/scalars/string)  
> 
> 
> The first line of the billing address.
> 

#### [`address_line2`](#)<Bullet />[`String`](/docs/api/types/scalars/string)  
> 
> 
> The second line of the billing address.
> 

#### [`card_brand`](#)<Bullet />[`String`](/docs/api/types/scalars/string)  
> 
> 
> The brand of the card. Null if the payment_type is not a card.
> 

#### [`city`](#)<Bullet />[`String`](/docs/api/types/scalars/string)  
> 
> 
> The city of the billing address.
> 

#### [`country`](#)<Bullet />[`String`](/docs/api/types/scalars/string)  
> 
> 
> The country of the billing address.
> 

#### [`exp_date`](#)<Bullet />[`String`](/docs/api/types/scalars/string)  
> 
> 
> The expiration date of the card. Null if the payment_type is not a card. Format: MMYY
> 

#### [`full_name`](#)<Bullet />[`String`](/docs/api/types/scalars/string)  
> 
> 
> The name on card or bank account.
> 

#### [`is_active`](#)<Bullet />[`Boolean`](/docs/api/types/scalars/boolean)  
> 
> 
> Whether or not the payment method token is active.
> 

#### [`last_four`](#)<Bullet />[`String`](/docs/api/types/scalars/string)  
> 
> 
> The last four digits of the card or bank account number.
> 

#### [`merchant_uid`](#)<Bullet />[`String`](/docs/api/types/scalars/string)  
> 
> 
> The Pay Theory unique identifier assigned to the merchant that the payment_method_token belongs to.
> 

#### [`metadata`](#)<Bullet />[`AWSJSON`](/docs/api/types/scalars/awsjson)

> Any additional data that was stored with the payment method token.
