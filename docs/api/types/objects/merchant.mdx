Merchants are the entities that are using Pay Theory to accept payments.


```graphql
type Merchant {
  ach_active: Boolean
  api_key: String
  card_active: Boolean
  cash_active: Boolean
  fee_model: FeeModel
  is_system: Boolean
  merchant_name: String
  merchant_uid: String
  parent_merchant_uid: String
  settings: MerchantSettings
  submitted_onboarding: Boolean
}
```
