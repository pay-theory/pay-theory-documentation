Transactions are a data object that can represent a payment, failed or successful, or a refund.


```graphql
type Transaction {
  account_code: String
  ach_return_details: AchReturnDetails
  additional_purchase_data: AdditionalPurchaseData
  authorization_id: String
  currency: String
  device_id: String
  dispute_status: DisputeStatus
  failure_reasons: [String]
  fee_mode: FeeMode
  fees: Int
  invoice: Invoice
  is_settled: Boolean
  gross_amount: Int
  merchant_uid: String
  metadata(
    query_list: [QueryPair]
  ): AWSJSON
  net_amount: Int
  parent_id: String
  payment_method(
    query_list: [QueryPair]
  ): PaymentMethodToken
  recurring: RecurringPayment
  reference: String
  refund_reason: RefundReason
  refund_voidable: Boolean
  refunded_amount: Int
  sale_id: String
  settlement_batch: Int
  status: TransactionStatus
  timezone: String
  transaction_date: AWSDateTime
  transaction_id: String
  transaction_type: TransactionType
  updated_row_at: AWSDateTime
}
```
