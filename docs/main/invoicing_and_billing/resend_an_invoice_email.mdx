# Resend an Invoice Email
The guide below will resend an invoice's email.

## Request
Insert the appropriate invoice ID into the request below to resend the invoice's email.

```graphql title="GraphQL"
mutation {
    createInvoiceEmail(
        invoice_id: "" # Insert invoice_id 
)}
```

:::note  Additional Parameters
View the extensive list of parameters for resending an invoice email in our API. 

<a href= "../../api/invoice#resend-an-invoice-email" class="button button--primary button--md">View All Parameters</a>
:::

***

## Return
The response will include a boolean value that will be true if the email was sent successfully.

```JSON title="JSON"
{
    "data": {
        "createInvoiceEmail": true
    }
}
```
