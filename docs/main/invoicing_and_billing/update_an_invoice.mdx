# Update an Invoice
Updating an invoice involves making changes to the billing or payment information included in the invoice record. The following call will update an existing invoice.

```graphql title="GraphQL"
mutation {
    updateInvoice(invoice_id: string, invoice_update_input:{
        invoice_description: string,
        currency: "USD",
        due_by: "2022-09-19"

    }) 
}
```

## Return
If the invoice is updated, the call will return true.

```JSON title="JSON"
{
    "data": {
        "updateInvoice": true
    }
}
```
