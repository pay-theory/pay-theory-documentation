# Record an Offline Payment
 Offline transactions are used to track payments that are made outside Pay Theory towards an Invoice. This guide will demonstrate how to create an offline transaction for an invoice.

## Request
In this example the payor has made a cash payment offline for $100.00. Insert the correlating `invoice_id` into the code below to record an offline payment.

```graphql title="GraphQL"
mutation {
    createOfflineTransaction(input:  {
        amount: 10000  # Amount in pennies
        invoice_id: "" # Insert invoice_id
        type: CASH, # Reference OfflineTransactionType
        transaction_date: "2022-09-19"

    }){
        amount
        type
    }
}
```

## Return
The response will contain the offline transaction object that was created.

```JSON title="JSON"
{
    "data": {
        "createOfflineTransaction": {
            "amount": 10000,
            "type": CASH,
        }
    }
}

```
